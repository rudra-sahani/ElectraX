# ElectraX

Premium Electronics E-Commerce Website built using React and Vite.

## Features

- Home Page
- Products Page
- Product Details Page
- Cart Page
- About Page
- Contact Page

## Tech Stack

- React
- Vite
- React Router DOM
- CSS
